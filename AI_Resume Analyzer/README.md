# AI Resume Analyzer

## Overview
AI Resume Analyzer is a Streamlit-based web application that analyzes resumes and detects technical skills.

## Features
- Resume PDF upload
- Skill extraction
- Resume scoring
- Suggestions for improvement

## Technologies Used
- Python
- Streamlit
- NLP
- pdfplumber

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
