Finan - AI Financial Advisor for India 💸

An AI Powered Financial Advisor Model that learns from Indian financial dataset to:
Predict user's current financial state 💰
Suggest monthly budget for achieving a better financial situation. 📈

Tech Stack 👩🏻‍💻:
Numpy, Pandas: Data Analysis and Dataframes

Scikit-learn: SVC model for categorical predictions, Random Forest Regressors for predicting monthly budget suggestions

Langextract: Extracting user's financial info from user query

Gemini API: Generating text response containing financial advice for user

Flask: For integrating web application interface to my ML Model Jupyter Notebook

Firebase: For user authentication and cloud storage for recent chats and saved budgets 

Project X-factor 🌟:
Training data, model predictions and financial advice for user, all are based on Indian economic landscape.

Security 🔐:

Implemented Firebase Auth and secure token verification middleware, reducing data latency by structuring user chat history tracking via cloud storage.

Engineered production-grade security architectures by enforcing Google Cloud API domain restrictions and managing secrets via Render environment variables to prevent credential exposure.
